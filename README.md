YfwHome 介绍
=================
1、本程序基于GatewayWorker开发的物联网控制系统 

YfwHome 版本 
=================
 版本1.0
 
 功能：
* 1、设备接入
* 2、客户端app接入
* 3、设备客户端绑定功能
 


程序启动与停止
=================
php start.php start  启动程序
php start.php start -d  后台启动程序 
php start.php stop   结束程序
